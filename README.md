# IT1214-Student-Grade-System

### Student Information
- **Reg.No:** 2023/Ict/094
- **Course:** IT1214 Object Oriented Programming

### Project Description
A console-based Java application to manage student grades. Users can add students, display all records, search by ID, and calculate class average. Input validation ensures marks are between 0-100.

### How to Run
1. Compile: `javac Student.java StudentGradeManagementSystem.java`
2. Run: `java StudentGradeManagementSystem`

### Features Implemented
- Add student with input validation
- Display all students with grades
- Search student by ID
- Calculate class average
- Grade calculation: A(90+), B(75-89), C(60-74), D(50-59), F(<50)
